● To pull a branch from GitHub:

  Fetch all remote branches first:
  git fetch origin

  Then either:

  Switch to an existing remote branch:
  git checkout branch-name

  Or explicitly create a local branch tracking the remote one:
  git checkout -b branch-name origin/branch-name

  To see all available remote branches:
  git branch -r
